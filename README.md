### Project Description
This is a web-based platform that allows students to book campus resources such as study rooms, labs, and tutoring sessions. Staff and admins can manage resources and monitor usage.

### Tech Stack
- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB
- DevOps: Docker, Docker Compose
- Tools: Jira, TestRail, LaTeX

### Features
- User authentication (student/staff/admin)
- Resource browsing
- Booking system
- Check-in system
- Admin dashboard
